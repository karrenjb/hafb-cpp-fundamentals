## Terminal Commands ($ means terminal prompt)
```bash
$ ls	#list folder
$ cd <dest>	# change directory
$ cd..		# parent directory
$ pwd		# show present working directory
$ rm <name>	# remove or delete file or director
System
. (one dot)	//present working directory
.. (two dots)	//parent directory
```
